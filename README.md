🍽️ Swiggy Sales Analysis using Python

📌 Project Overview

This project focuses on analyzing Swiggy sales data to understand customer ordering behavior, sales performance, and revenue trends using Python. The analysis includes data cleaning, KPI calculation, exploratory data analysis (EDA), and insightful visualizations that help identify business opportunities and support data-driven decision-making.

🎯 Objectives
Analyze overall sales performance.
Calculate important business KPIs.
Identify sales trends over time.
Discover top-performing cities and states.
Analyze food category performance.
Evaluate customer ratings and order values.

📊 Key KPIs
Total Sales
Average Rating
Average Order Value
Total Orders
Quarterly Sales Summary

📈 Visualizations
Monthly Sales Trend
Weekly Sales Trend
Daily Sales Trend
Top 5 Cities by Sales
State-wise Sales Analysis
Food Category Revenue Analysis
Quarterly Sales Performance
